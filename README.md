# COVID-19 Data Analysis (Excel)

## 📌 Overview
This project focuses on cleaning and analyzing COVID-19 dataset using Microsoft Excel.

## 🛠 Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Conditional Formatting

## 📊 Key Insights
- Calculated death %, recovery rate, infection rate
- Categorized countries into risk levels
- Compared continent-wise trends

## 🎯 Objective
To convert raw data into meaningful insights and improve data analysis skills.
